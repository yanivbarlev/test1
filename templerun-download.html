Temple Run

Category: Games

Price: Free (with in-app purchases)

Developer: Imangi Studios

Description

Temple Run is an endless runner game that puts you in the shoes of an explorer who has stolen a cursed idol from an ancient temple. Now you must run, jump, and slide to escape the demonic monkey chasing you. Collect coins, power-ups, and unlock new characters as you try to survive as long as possible in this addictive mobile game.

  • Multiple character options
  • Various power-ups and abilities
  • Regular updates with new content
  • Global leaderboards
  • Daily challenges and missions
  • Character customization
  • Multiple environments to run through
  • Achievement system
  • Social sharing features
