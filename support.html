Getting Started

How do I install Text Polish?
1. Visit the Chrome Web Store
2. Click "Add to Chrome"
3. Pin the extension to your toolbar for easy access
4. Add your API key in the settings to get started
How do I get an API key?
For Groq (Recommended - Free tier available):
1. Visit console.groq.com
2. Sign up for a free account
3. Go to API Keys section
4. Create a new API key
5. Copy and paste it into Text Polish settings

For OpenAI (Paid service):
1. Visit platform.openai.com
2. Sign up and add billing information
3. Go to API Keys section
4. Create a new API key
5. Copy and paste it into Text Polish settings

Frequently Asked Questions

How do I use Text Polish?
Method 1 - Right-click menu:
1. Highlight any text on any website
2. Right-click and select "Polish Selected Text with AI"
3. Choose your preferred style or action

Method 2 - Extension popup:
1. Click the Text Polish icon in your toolbar
2. Type or paste your text
3. Select Enhance, Explain, or Translate
4. Copy the improved text
What writing styles are available?
• Professional: Perfect for business emails and formal documents
• Casual: Great for friendly communications and social media
• Academic: Ideal for research papers and scholarly writing
• Creative: Adds flair and creativity to your writing
• Concise: Makes your text shorter and more direct

Is my data secure?
Yes! Text Polish prioritizes your privacy:
• Your text is only sent to AI services when you explicitly request enhancement
• API keys are stored securely on your device
• Text responses are cached locally for 1 hour, then automatically deleted
• We never store your content on our servers
• All communications use HTTPS encryption
• Read our full Privacy Policy
Why am I getting API errors?
Common solutions:
• Check that your API key is entered correctly
• Verify your API account has available credits/quota
• Try switching between Groq and OpenAI in settings
• Check your internet connection
• Contact us if the problem persists
What are the keyboard shortcuts?
• Ctrl+Shift+P (Windows/Linux) or Cmd+Shift+P (Mac): Open Text Polish popup
• Ctrl+Shift+E (Windows/Linux) or Cmd+Shift+E (Mac): Enhance selected text
The extension isn't working on some websites. Why?
Some websites (like chrome:// pages, extension pages, or certain secure sites) restrict extension functionality for security reasons. This is normal browser behavior. The extension works on most regular websites including Gmail, Google Docs, social media, news sites, etc.

Troubleshooting

Text Polish isn't appearing in my right-click menu
1. Make sure you have text selected before right-clicking
2. Try refreshing the page
3. Check if the extension is enabled in chrome://extensions/
4. Try restarting your browser
The extension popup is blank or not loading
1. Try clicking the extension icon again
2. Check your internet connection
3. Disable and re-enable the extension
4. If the problem persists, try removing and reinstalling the extension
I'm getting "Rate limit exceeded" errors
This means you've reached your API provider's usage limits:
• For Groq: Check your free tier limits or upgrade your plan
• For OpenAI: Add more credits to your account
• Try again in a few minutes
• Consider switching to the other API provider as a backup

Advanced Tips

Can I use both Groq and OpenAI?
Yes! You can add both API keys in the settings. Text Polish will use Groq as the primary service and automatically fall back to OpenAI if Groq is unavailable. This provides better reliability.
How can I make Text Polish work faster?
• Use Groq API (generally faster than OpenAI)
• Keep your text selections reasonably short
• Text Polish caches responses, so repeated requests are instant
• Ensure you have a stable internet connection
